# SPTI · 运动英雄十五式

**Sport Personality Type Indicator** — 一个娱乐向的运动人格测评工具。

灵感来自 MBTI / SBTI，为运动人格而设计。20 道题，5 分钟，找到你的运动英雄和当前段位。

## 🎮 如何开始

打开 [index.html](./index.html) 即可开始测试。

## 📊 测试维度

- **VO2max 天花板**（心肺上限）
- **有氧基础**（耐力续航）
- **爆发力**（瞬时功率）
- **肌肉耐力**（中时长力量输出）
- **恢复力**（睡眠 + 训练恢复）

## 🏆 15 个运动英雄 × 5 个段位

- **躺平档** · 家里蹲鲁班七号 / 脆皮大学生小乔 / 久坐工位猪八戒 / 周末战神孙策 / 减肥轮回周瑜
- **英雄档** · 续航伽罗 / 坦克牛魔 / 数据党扁鹊 / 均衡赵云 / 爆发李白 / 养生瑶 / 研究诸葛亮 / 游击裴擒虎
- **隐藏档** · 苦行僧花木兰 / 赛博斯巴达关羽

每个英雄都有 5 段位（青铜 → 白银 → 黄金 → 钻石 → 王者）。

## ⚠️ 免责声明

本测试仅供娱乐与自我认知，不构成医学或运动医学建议。参考了 IPAQ、Jackson PA-R、ACSM、TQR 等问卷的题项设计，但不是严谨的学术量表。
